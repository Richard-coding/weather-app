# 🌦️ Weather App

A modern weather application built with React that provides real-time weather data for any city using the Open-Meteo API.
The project was created to practice frontend architecture, API integration, state management, asynchronous requests, and responsive UI development using modern React concepts.

# 📚 About The Project

This application allows users to search for weather information from cities around the world in real time.

The main goal of the project was not only to display weather data, but also to simulate a more realistic frontend application architecture using reusable components, API abstraction, loading states, error handling, and responsive design principles.

This project was developed as part of my frontend learning journey while studying React and modern JavaScript.

# 📂 Project Structure

```bash
src/
│
├── components/
├── services/
├── hooks/
├── assets/
├── styles/
└── App.jsx
```

### Structure Explanation

* `components/` → reusable UI components
* `services/` → API request abstraction
* `hooks/` → custom React hooks
* `styles/` → global and component styles
* `assets/` → static files and images

# ⚙️ Getting Started

## Clone the repository

```bash
git clone https://github.com/Richard-coding/weather-app
```

## Enter the project folder

```bash
cd weather-app
```

## Install dependencies

```bash
npm install
```

## Run the development server

```bash
npm run dev
```

The application will be available at:

```bash
http://localhost:5173
```

# 🌍 API Integration

This project uses the Open-Meteo API to retrieve weather information and geolocation data.

The application workflow is:

1. User searches for a city
2. The geocoding API retrieves city coordinates
3. Weather API fetches real-time weather data
4. The UI updates dynamically with the received data
