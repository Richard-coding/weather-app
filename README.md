# 🌦️ Weather App

A weather application built with React that allows users to search for real-time weather data for any city using the Open-Meteo API.

---

## 🚀 Live Demo

👉 Access the project: https://weather-app-hazel-theta-69.vercel.app/

---

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS
- Open-Meteo API

---

## ⚙️ Features

- 🔍 Search weather by city name
- 🌡️ Display current weather conditions
- ⏳ Loading state during API requests
- ⚠️ Error handling for failed requests
- 📱 Responsive design for different screen sizes

## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/Richard-coding/weather-app

# Navigate to the project folder
cd weather-app

# Install dependencies
npm install

# Run the development server
npm run dev
```

---

## 🏗️ Production Build

```bash
npm run build
npm run preview
```
